# VHD / VHDX — Microsoft Virtual Hard Disk

> Formats: `.vhd`, `.vhdx`

---

## Overview

Microsoft's virtual hard disk formats. Read by **libvhdi**, part of the libyal project.

---

## Library

| Library | Source |
|---|---|
| libvhdi | https://github.com/libyal/libvhdi |

libvhdi shares the same libyal common dependencies as libewf. Note: VHDX support is partial — check the GitHub issues for current status.
